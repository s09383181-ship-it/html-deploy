How it works

Your HTML is uploaded as a new file in the html-deploy GitHub repo, then served by GitHub Pages. Each deploy gets a unique URL like .../sites/abc123/.

After clicking Deploy, the app waits up to 60 seconds for GitHub Pages to actually serve the new file (it usually takes 10-30s). Once verified, the link is shown.

Token storage: Your GitHub token is saved only in this browser's localStorage and used directly to talk to GitHub's API.
